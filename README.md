# html-skelton

Adds the following snippet as `html-skelton`

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>${TM_FILENAME_BASE}</title>
    <link rel="icon" type="image/x-icon" href="./assets/favicon.ico" />
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
  </head>

  <body></body>
</html>
```
